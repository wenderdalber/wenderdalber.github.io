Sobre

Apaixonado por tecnologia, desenvolvedor web a três anos na plataforma .NET, iniciante em Ruby on Rails

  • 2009-2012

    Formação em Tecnólogo de Informática - Enfâse em banco de dados

    Em 2009 iniciei o curso de Tecnologia da Informação na Fatec Guaratingueta. Tive contato com tecnologias Java e .NET de desenvolvimento, além da enfâse em banco de dados trabalhando com Oracle e MySQL!

  • 2013-2014

    Especialização em Projeto e Desenvolvimento de Aplicações WEB

    Em março de 2013 iniciei uma especialização em projetos e desenvolvimento de aplicações WEB na Fatea de Lorena. Desenvolvimento de aplicações robustas, testes de carga e unitários. Controle de versão com GIT e desenvolvimento de aplicações em .NET e Grails.

  • Fevereiro-Setembro 2014

    Desenvolvedor Web Plano Digital

    Desenvolvedor web na Agência Plano Digital. Desenvolvimento de web sites para Apex Brasil, Sodexo, Caixa. Foco em aplicações .NET, banco de dados Oracle e SQL Server. Manutenção de servidores web da empresa, gerenciamento e controle de versão com git.

  • Outubro 2014

    Fundação Betaup

    Fundação e inicio dos trabalhos da Betaup, ao lado de Felipe Rocha e Daniele Fernandes. Foco no desenvolvimento do projeto Click Hotel Web, voltado para o gerenciamento de hotéis da região de Aparecida-SP.
